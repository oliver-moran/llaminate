Module: MimeTypes

Static MIME types for various file formats.
Properties:
Name Type Description
Llaminate.GIF string MIME type for GIF images (`image/gif`).
Llaminate.IMAGE string MIME type for unknown images (`image/*`).
Llaminate.JPEG string MIME type for JPEG images (`image/jpeg`).
Llaminate.PDF string MIME type for PDF files (`application/pdf`).
Llaminate.PNG string MIME type for PNG images (`image/png`).
Llaminate.WEBP string MIME type for WEBP images (`image/webp`).

Example

const response = await mistral.complete("Summarize the content of this image.", {
 attachments: [ { type: Llaminate.JPEG, url: "https://example.com/image.jpg" } ]
});
